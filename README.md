# 🧵 GALILEO-ITA (Analizzatore Difetti Tessuti)

[![Java](https://img.shields.io/badge/Java-ED8B00?style=for-the-badge&logo=openjdk&logoColor=white)](https://www.java.com/)
[![Ollama](https://img.shields.io/badge/Ollama-AI-blue?style=for-the-badge)](https://ollama.ai/)
[![Swing](https://img.shields.io/badge/UI-Swing-orange?style=for-the-badge)](https://docs.oracle.com/javase/tutorial/uiswing/)

Progetto sviluppato durante il percorso di **PCTO** (Alternanza Scuola-Lavoro) presso **Galileo Italia SRL**. Si tratta di un software innovativo che utilizza l'intelligenza artificiale per identificare e analizzare difetti nei tessuti.

## 🚀 Caratteristiche
- **AI-Powered**: Utilizza il modello **Gemma 3:4b** tramite Ollama per l'analisi intelligente.
- **Dual Interface**: Offre sia un'interfaccia grafica (GUI) moderna in Swing che una potente interfaccia a riga di comando (CLI).
- **Integrazione Dati**: Gestione dei dati tramite Jackson per il parsing JSON.
- **Analisi in Tempo Reale**: Progettato per supportare flussi di lavoro rapidi nell'industria tessile.

## 🛠️ Tecnologie
- **Java** (JDK 11+)
- **Ollama** (Gemma AI)
- **Jackson** (JSON handling)
- **NetBeans / Ant** (Build system)

## 📦 Installazione
1. Assicurati di avere [Ollama](https://ollama.ai/) installato e il modello `gemma:3b` scaricato.
2. Clona la repository:
   ```bash
   git clone https://github.com/St0rmosu/GALILEO-ITA.git
   ```
3. Compila ed esegui tramite NetBeans o Ant.
